## Paso 4: Nivel 3 – Prompt con few-shot

Este prompt incluye un ejemplo de formato deseado. Primero, **copia este ejemplo:**

```python
def es_palindromo(text: str) -> bool:
    """
    Verifica si un texto es un palíndromo.
    
    Args:
        text: String a verificar
    
    Returns:
        True si es palíndromo, False si no
    """
    cleaned = text.replace(" ", "").lower()
    return cleaned == cleaned[::-1]
```

**Ahora copia y pega en cada IA este prompt completo:**

## Conclusión

La calidad de un prompt determina la calidad del resultado. Invertir 2 minutos en escribir un prompt mejorado (Nivel 2) produce código que necesitaría 20 minutos de refactorización si lo generaras desde Nivel 1. Con few-shot (Nivel 3), el código muchas veces está **listo sin cambios**.

Este es el poder del prompt engineering.
