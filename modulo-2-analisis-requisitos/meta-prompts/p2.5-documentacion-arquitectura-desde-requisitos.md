# P2.5 – GENERACIÓN DE DOCUMENTACIÓN DE ARQUITECTURA DESDE REQUISITOS

Actúa como un diseñador instruccional experto en formación de desarrolladores de software. Necesito que generes un ejercicio práctico detallado para un curso de IA aplicada al desarrollo de software.

---

## Paso 1: Introducción

La arquitectura del software puede y debería inferirse directamente de los requisitos bien redactados. Una especificación ejecutable no solo describe "qué" hace el sistema, sino también "cómo" se estructura y comunica internamente. Esto permite que una IA genere documentación arquitectónica inicial como diagramas C4 o registros de decisiones arquitectónicas (ADR) de forma automatizada, cerrando el ciclo entre análisis y diseño.
