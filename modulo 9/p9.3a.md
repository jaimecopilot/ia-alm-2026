Actúa como un arquitecto de ALM Generativo experto en la fase de diseño de software con herramientas visuales. Estás diseñando un laboratorio práctico para desarrolladores que quieren usar herramientas de diseño visual con IA (Claude Artifacts, ChatGPT + Canvas, Draw.io + Sparkle AI, Gemini Code Wiki) para generar diagramas de arquitectura C4 y UML.

## Contexto del ALM Generativo

En el ALM generativo, el diseño no es una fase aislada: los diagramas se generan desde las especificaciones ejecutables creadas en la fase de análisis, y a su vez alimentan la generación de código en la fase de implementación. Las herramientas visuales con IA permiten crear diagramas como código (Mermaid, PlantUML) que se versionan en el repositorio y evolucionan con el sistema.

## Tu tarea

Genera un laboratorio guiado "Paso a Paso" que el alumno pueda ejecutar con cualquiera de estas herramientas: Claude Artifacts, ChatGPT + Canvas, Draw.io + Sparkle AI o Gemini Code Wiki.

El laboratorio debe cubrir:

### Visión en el ALM
Explica que los diagramas generados por IA no son imágenes estáticas, sino artefactos vivos que se regeneran cuando el código cambia y se versionan en Git.

### Técnica 1: Generar diagrama C4 de contexto
Guía al alumno para que, partiendo de una especificación ejecutable, pida a la IA que genere un diagrama C4 de contexto en Mermaid mostrando actores, sistemas externos y el sistema principal.

### Técnica 2: Generar diagrama C4 de contenedores
Guía al alumno para que genere el nivel de contenedores, mostrando aplicaciones, bases de datos y servicios con sus relaciones y protocolos.

### Técnica 3: Generar diagrama de clases UML
Guía al alumno para que genere un diagrama de clases con entidades, atributos, métodos y relaciones.

### Técnica 4: Generar diagrama de secuencia
Guía al alumno para que genere un diagrama de secuencia para un flujo principal (ej: reserva de sala), incluyendo fragmentos condicionales para casos de error.

### Técnica 5: Iteración visual conversacional
Guía al alumno para que refine los diagramas mediante prompts sucesivos ("añade un balanceador de carga", "muestra la relación con el servicio de notificaciones") y observe cómo la IA actualiza el diagrama.

## Conexión con el resto del ALM

Guía al alumno para que guarde los diagramas generados como archivos Markdown en el repositorio, simulando cómo se versionarían y se conectarían con la fase de implementación.
