## Tu tarea

Genera un laboratorio guiado "Paso a Paso" que el alumno pueda ejecutar con ChatGPT.

El laboratorio debe cubrir:

### Visión en el ALM
Explica cuándo usar un chat generalista frente a un asistente de IDE en la fase de desarrollo.

### Técnica 1: Generación de CRUD desde especificación
Guía al alumno para que pegue una especificación ejecutable y pida a ChatGPT que genere el código completo.

### Técnica 2: Refactorización aplicando patrones
Guía al alumno para que pegue código y pida refactorizarlo aplicando un patrón específico.

### Técnica 3: Traducción entre lenguajes
Guía al alumno para que pegue código en un lenguaje y pida traducirlo a otro.

### Técnica 4: Explicación de código complejo
Guía al alumno para que pegue código ofuscado y pida una explicación línea por línea.

### Técnica 5: Generación de tests
Guía al alumno para que pida tests unitarios para el código generado.

## Conexión con el resto del ALM

Pide al alumno que compare el código generado por ChatGPT con el generado por un asistente de IDE.
