# Módulo 9 — Herramientas y Flujos para un ALM Generativo Real

## Visión general

Este módulo introduce el uso práctico de herramientas de Inteligencia Artificial aplicadas al ciclo de vida del desarrollo de software (ALM/SDLC).

A diferencia de módulos anteriores centrados en técnicas o fases aisladas, aquí el objetivo es comprender cómo múltiples herramientas de IA colaboran dentro de un flujo completo de ingeniería de software.

El alumno aprenderá a diseñar, ejecutar y evaluar flujos completos de ALM generativo combinando:

- Chats generalistas
- Asistentes de IDE
- Agentes autónomos
- Herramientas DevOps
- Plataformas cloud
- Sistemas de gobernanza

---

## Objetivos de aprendizaje

Al finalizar el módulo, el alumno será capaz de:

- Identificar el rol de cada categoría de herramienta en el ALM generativo
- Diseñar flujos completos desde requisitos hasta despliegue
- Usar IA para generar artefactos reales de software (código, tests, pipelines, documentación)
- Evaluar la calidad de outputs generados por IA
- Aplicar gobernanza básica sobre sistemas generativos

## Metodología del módulo

Cada laboratorio sigue el mismo modelo:

1. **Contexto del ALM**
2. **Objetivo del ejercicio**
3. **Entrada (input del alumno)**
4. **Ejecución guiada con IA**
5. **Artefacto generado (output obligatorio)**
6. **Validación (checklist técnico)**
7. **Reflexión estructurada**

---

## Principio clave

> En el ALM generativo, el valor no está en el prompt, sino en el artefacto verificable que se genera.

Por ello, cada ejercicio debe producir al menos uno de los siguientes:

- Código funcional
- Tests ejecutables
- Pipeline CI/CD
- Especificación estructurada
- Diagramas versionables
- Configuración cloud

---

## Evaluación

El alumno será evaluado según:

- Correctitud del artefacto generado
- Consistencia con la especificación
- Aplicación de patrones de diseño
- Uso correcto de IA en cada fase
- Capacidad de iteración y mejora

---

## Resultado esperado del módulo

Al finalizar, el alumno será capaz de simular un flujo completo de desarrollo software asistido por IA, desde la idea inicial hasta el despliegue y mantenimiento.

---

## Nota pedagógica

Este módulo no enseña herramientas aisladas.

Enseña cómo diseñar sistemas de producción de software asistidos por IA.
