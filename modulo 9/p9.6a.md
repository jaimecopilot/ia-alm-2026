El desarrollador configura las reglas y supervisa.

## El laboratorio debe cubrir
Visión en el ALM: Explica que DevOps con IA cierra el círculo: el código generado se prueba, se construye y se despliega automáticamente.

Técnica 1: Generar pipeline CI/CD con lenguaje natural: Guía al alumno para que describa su pipeline y use Copilot Chat o GitLab Duo para generar la configuración YAML.

Técnica 2: Usar Agentic Workflows (gh aw): Guía al alumno para que cree un archivo Markdown con la descripción del workflow y lo compile con gh aw.

Técnica 3: Generar pipeline desde cero con GitLab Duo CI Expert: Guía al alumno para que use el CI Expert Agent en GitLab.

Técnica 4: Diagnosticar fallos de CI: Guía al alumno para que provoque un fallo y use la IA para diagnosticarlo.

Técnica 5: Self-Healing CI: Guía al alumno para que configure un workflow que detecte fallos y cree issues automáticamente.
